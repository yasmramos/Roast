# Roast Language

**Roast**: *Roast the unnecessary.* A modern, concise, and powerful programming language designed for the JVM.

## Overview

Roast is a statically-typed language that compiles directly to JVM bytecode using ANTLR4 for parsing and ASM for code generation. It combines modern syntax features with seamless Java interoperability.

## Features

- **Modern Syntax**: Clean, readable syntax inspired by Kotlin and Swift.
- **Type Inference**: Smart type deduction reduces boilerplate (`val x = 10` infers `Int`).
- **JVM Native**: Compiles directly to optimized `.class` files.
- **Control Flow**: Supports `if/else`, `while`, and logical operators.
- **Functions**: First-class functions with default arguments and named parameters.
- **Classes & Objects**: Object-oriented programming with inheritance and interfaces.
- **Null Safety**: Designed with null safety in mind (WIP).

## Requirements

- **JDK 17** or higher
- **Gradle** (Wrapper included)

## Building the Project

```bash
./gradlew clean build
```

## Usage

### Compile a Roast file

```bash
./gradlew run
```

This will compile all `.roast` files in the `examples/` directory and output `.class` files to `build/generated/roast`.

### Run the generated bytecode

```bash
java -cp build/generated/roast io.github.yasmramos.roast.examples.HelloWorld
```

## Example Code

### Hello World

```roast
fun main() {
    println("Hello, Roast!")
}
```

### Variables and Control Flow

```roast
fun main() {
    val name = "Developer"
    var count = 0
    
    if (count < 5) {
        println("Counting up...")
        while (count < 5) {
            println(count)
            count = count + 1
        }
    } else {
        println("Count is high")
    }
}
```

### Functions and Types

```roast
fun add(a: Int, b: Int): Int {
    return a + b
}

fun greet(name: String) {
    println("Hello, " + name)
}
```
